# **伪随机数**

作者: 臧珠萍

时间限制: 1s

章节: 一维数组

###### **问题描述**

计算机经常被用来生成伪随机数的序列。一种常见的伪随机数生成技术被称为线性同位素法。如果上一个生成的伪随机数是L，那么下一个数字由(Z\*L+I)mod M计算所得 ，其中Z是一个常数乘数，I是一个常数增量，M是一个常数模。例如，假设Z是7，I是5，而M是12。如果第一个随机数（通常称为种子）是4，那么我们可以确定接下来的几个伪随机数如下表所示：

!\[题目配图](./image.jpg)

正如你所看到的，表中的伪随机数序列在六个数字之后重复。很清楚，用这种技术可以生成的最长序列受到模数M的限制。

在这个问题中，你将得到几组Z、I、M和种子L的数值，每组数值都不超过四位。对于每一组数值，你要确定将产生的伪随机数周期的长度。但是要小心：这个周期可能不是从种子开始的。

###### **输入说明**

每个输入行将包含四个整数值，依次为Z、I、M和L，保证L小于M。最后一行将包含四个零，标志着输入数据的结束。

其中Z、I、M和L的范围都在\[1,9999]之间。

###### **输出说明**

对于每个输入行，显示案例编号比如“Case 1:”（它们是按顺序编号的，从1开始）以及伪随机数序列周期的长度。

###### **输入范例**

7 5 12 4

5173 3849 3279 1511

0 0 0 0

###### **输出范例**

Case 1: 6

Case 2: 546
